
--------------- Import all table of this site database ----------------

Follow these steps:-

1- Open your phpMyAdmin.
2- Click on 'New' and create new datebase name 'warraich-traders'.
3- Click on datebase that you created.
4- Click on 'Import' which is present on the top menu.
5- Click on 'Choose File' and select the file which is present in 
    warraich-traders/database/warraich-traders.sql
6- And press 'open'.

All of these table that present in sql file is imported in your database.
